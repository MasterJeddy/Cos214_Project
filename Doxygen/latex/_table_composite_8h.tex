\doxysubsection{Detailed Description}
This file contains the declaration of the \doxylink{class_table_composite}{Table\+Composite} class which acts here as the the Composite participant in the Composite design pattern. For example when taking an order, the waiter will interact with an object from this class. This class implements the interface provided by the \doxylink{class_table_component}{Table\+Component} interface. It contains either children that are either composites(\+Table objects) or leaves (\doxylink{class_customer}{Customer} objects). This allows for us to add tables to each other (to make space for larger groups), and \doxylink{class_customer}{Customer} objects may be added to tables.
